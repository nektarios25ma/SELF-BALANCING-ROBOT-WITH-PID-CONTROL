Εδώ γίνεται  μια προσπάθεια να γίνει μια εισαγωγή και δοκιμή αυτής της μεθόδου PID  σε ένα αυτοεξισορροπούμενο ρομποτάκι το οποίο θα ισορροπεί στις δύο ρόδες και θα ακολουθεί τις εντολές μας.
